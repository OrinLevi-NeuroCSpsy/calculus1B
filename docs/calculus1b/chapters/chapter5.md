# פרק 5: פונקציות במשתנה אחד

## 5.1 הגדרות בסיסיות

!!! info "הגדרה 5.1"
    **פונקציה ממשית** היא $f : D \to \mathbb{R}$ כאשר $D \subseteq \mathbb{R}$ **תחום ההגדרה**.

!!! info "הגדרה 5.2 (סוגי פונקציות)"
    - **חסומה מלעיל:** קיים $M$ כך ש-$f(x) \le M$ לכל $x \in D$
    - **חסומה מלרע:** קיים $m$ כך ש-$f(x) \ge m$ לכל $x \in D$
    - **עולה:** $x_1 < x_2 \Rightarrow f(x_1) \le f(x_2)$
    - **עולה ממש:** $x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$
    - **זוגית:** $f(-x) = f(x)$
    - **אי-זוגית:** $f(-x) = -f(x)$
    - **מחזורית:** קיים $T > 0$ כך ש-$f(x + T) = f(x)$

## 5.2 פונקציות אלמנטריות

!!! note "הפונקציות האלמנטריות"
    1. **פולינומים:** $p(x) = a_n x^n + \cdots + a_1 x + a_0$
    2. **פונקציות רציונליות:** $\frac{p(x)}{q(x)}$
    3. **טריגונומטריות:** $\sin, \cos, \tan, \cot$
    4. **טריגונומטריות הפוכות:** $\arcsin, \arccos, \arctan$
    5. **מעריכית:** $e^x$, $a^x$
    6. **לוגריתמית:** $\ln x$, $\log_a x$
    7. **חזקה כללית:** $x^a = e^{a \ln x}$

## 5.3 פונקציות מיוחדות

!!! info "פונקציית דיריכלה"
    $$D(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

    פונקציה זו **אינה רציפה בשום נקודה!**

!!! info "פונקציית הסימן"
    $$\text{sgn}(x) = \begin{cases} 1 & x > 0 \\ 0 & x = 0 \\ -1 & x < 0 \end{cases}$$

!!! info "פונקציית הרצפה"
    $$\lfloor x \rfloor = \max\{n \in \mathbb{Z} : n \le x\}$$

## תרגילים

!!! question "תרגיל 1"
    הוכיחו כי לכל $f : \mathbb{R} \to \mathbb{R}$ קיימות $g$ זוגית ו-$h$ אי-זוגית כך ש-$f = g + h$.

??? success "פתרון"
    נגדיר:
    $$g(x) = \frac{f(x) + f(-x)}{2}, \quad h(x) = \frac{f(x) - f(-x)}{2}$$

    **בדיקה:**

    - $g(-x) = \frac{f(-x) + f(x)}{2} = g(x)$ — **זוגית** $\checkmark$
    - $h(-x) = \frac{f(-x) - f(x)}{2} = -h(x)$ — **אי-זוגית** $\checkmark$
    - $g(x) + h(x) = \frac{f(x) + f(-x) + f(x) - f(-x)}{2} = f(x)$ $\checkmark$

    $\blacksquare$
