# פרק 7: רציפות

## 7.1 הגדרת רציפות

!!! info "הגדרה 7.1"
    $f$ **רציפה בנקודה $x_0$** אם $\lim_{x \to x_0} f(x) = f(x_0)$.

    כלומר:
    $$\forall \varepsilon > 0 \; \exists \delta > 0 : |x - x_0| < \delta \Rightarrow |f(x) - f(x_0)| < \varepsilon$$

!!! info "הגדרה 7.2"
    $f$ **רציפה בקטע $I$** אם רציפה בכל נקודה של $I$.

!!! success "משפט 7.3 (אפיון היינה)"
    $f$ רציפה ב-$x_0$ $\iff$ לכל $x_n \to x_0$ מתקיים $f(x_n) \to f(x_0)$.

## 7.2 אריתמטיקה של פונקציות רציפות

!!! success "משפט 7.4"
    אם $f, g$ רציפות ב-$x_0$ אז גם:

    1. $f + g$
    2. $f \cdot g$
    3. $\frac{f}{g}$ (אם $g(x_0) \neq 0$)
    4. $f \circ g$

!!! success "משפט 7.5"
    כל פונקציה אלמנטרית רציפה בתחום הגדרתה.

## 7.3 משפט ערך הביניים (IVT)

!!! success "משפט 7.6 (IVT)"
    תהי $f : [a, b] \to \mathbb{R}$ רציפה. אם $f(a) < c < f(b)$ אז קיים $x_0 \in (a, b)$ כך ש-$f(x_0) = c$.

??? example "הוכחה (רעיון)"
    נגדיר $A = \{x \in [a, b] : f(x) < c\}$.

    $A$ לא ריקה וחסומה, לכן קיים $x_0 = \sup A$.

    מראים ש-$f(x_0) = c$ (אם $f(x_0) \neq c$ מגיעים לסתירה). $\blacksquare$

!!! example "יישום: קיום שורש"
    לכל פולינום ממעלה אי-זוגית יש שורש ממשי.

    **הוכחה:** $\lim_{x \to \infty} p(x) = \infty$, $\lim_{x \to -\infty} p(x) = -\infty$.

    קיימים $a, b$ כך ש-$p(a) < 0 < p(b)$. לפי IVT קיים שורש. $\blacksquare$

## 7.4 משפט ויירשטראס

!!! success "משפט 7.7 (ויירשטראס)"
    תהי $f : [a, b] \to \mathbb{R}$ רציפה. אז:

    1. $f$ **חסומה**
    2. $f$ **מקבלת מקסימום ומינימום**

## 7.5 רציפות במידה שווה

!!! info "הגדרה 7.8"
    $f$ **רציפה במידה שווה** אם:
    $$\forall \varepsilon > 0 \; \exists \delta > 0 \; \forall x, y : |x - y| < \delta \Rightarrow |f(x) - f(y)| < \varepsilon$$

!!! note "ההבדל"
    - **רציפות:** $\delta$ תלוי ב-$\varepsilon$ **וב-$x_0$**
    - **רציפות במידה שווה:** $\delta$ תלוי **רק ב-$\varepsilon$**

!!! success "משפט 7.9 (היינה-קנטור)"
    אם $f : [a, b] \to \mathbb{R}$ רציפה בקטע סגור וחסום, אז $f$ **רציפה במידה שווה**.

!!! example "דוגמה נגדית"
    $f(x) = \frac{1}{x}$ על $(0, 1]$ רציפה אך **לא** רציפה במידה שווה.

## תרגילים

!!! question "תרגיל 1"
    הוכיחו כי למשוואה $x^5 + x = 1$ יש פתרון יחיד ב-$[0, 1]$.

??? success "פתרון"
    נגדיר $f(x) = x^5 + x - 1$.

    **קיום:** $f(0) = -1 < 0$, $f(1) = 1 > 0$. לפי IVT, קיים $x_0 \in (0, 1)$ כך ש-$f(x_0) = 0$.

    **יחידות:** $f'(x) = 5x^4 + 1 > 0$ לכל $x$, לכן $f$ עולה ממש.

    פונקציה עולה ממש חוצה כל ערך לכל היותר פעם אחת. $\blacksquare$

!!! question "תרגיל 2"
    תהי $f : [0, 1] \to [0, 1]$ רציפה. הוכיחו שקיימת **נקודת שבת** $x_0$ כך ש-$f(x_0) = x_0$.

??? success "פתרון"
    נגדיר $g(x) = f(x) - x$.

    $g(0) = f(0) - 0 = f(0) \ge 0$

    $g(1) = f(1) - 1 \le 0$

    - אם $g(0) = 0$: $x_0 = 0$ נקודת שבת
    - אם $g(1) = 0$: $x_0 = 1$ נקודת שבת
    - אחרת: $g(0) > 0 > g(1)$, לפי IVT קיים $x_0 \in (0, 1)$ כך ש-$g(x_0) = 0$

    בכל מקרה $f(x_0) = x_0$. $\blacksquare$
