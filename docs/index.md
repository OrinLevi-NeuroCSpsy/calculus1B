## נושאים עיקריים לפי חלק

### חלק I - יסודות
| נושא | תיאור |
|------|-------|
| **תורת הקבוצות** | קבוצות, תת-קבוצות, איחוד, חיתוך, משלים, דה-מורגן |
| **פונקציות** | חח"ע, על, הפיכות, עוצמות |
| **המספרים הממשיים** | ערך מוחלט, חסמים, sup/inf |
| **אקסיומת השלמות** | תכונה ארכימדית, צפיפות הרציונליים |

### חלק II - סדרות וטורים
| נושא | תיאור |
|------|-------|
| **סדרות** | הגדרת גבול, חסימות, מונוטוניות |
| **משפטים מרכזיים** | סנדוויץ', בולצאנו-ויירשטראס, קושי |
| **limsup/liminf** | גבול עליון ותחתון |
| **טורים** | סכומים חלקיים, התכנסות |
| **מבחני התכנסות** | השוואה, מנה (דלמבר), שורש (קושי), אינטגרל, לייבניץ |
| **התכנסות בהחלט** | דיריכלה, אבל |

### חלק III - גבולות ורציפות
| נושא | תיאור |
|------|-------|
| **גבולות** | הגדרת $\varepsilon$-$\delta$, היינה, גבולות חד-צדדיים |
| **רציפות** | הגדרה, אריתמטיקה, הרכבה |
| **משפטים** | ערך הביניים (IVT), ויירשטראס |
| **רציפות במידה שווה** | היינה-קנטור |

### חלק IV - גזירות
| נושא | תיאור |
|------|-------|
| **נגזרות** | הגדרה, כללי גזירה, שרשרת |
| **משפטים** | פרמה, רול, לגרנז', קושי |
| **לופיטל** | צורות $\frac{0}{0}$, $\frac{\infty}{\infty}$, $0 \cdot \infty$, $1^\infty$, $0^0$ |
| **טיילור** | פולינום טיילור, שארית לגרנז', מקלורן |

### חלק V - אינטגרלים
| נושא | תיאור |
|------|-------|
| **פונקציות קדומות** | אנטי-נגזרת, טבלת אינטגרלים |
| **אינטגרל רימן** | סכומי דארבו, קריטריון רימן |
| **המשפט היסודי** | חלק I ו-II |
| **שיטות אינטגרציה** | החלפת משתנים, חלקים, שברים חלקיים |
| **אינטגרלים לא אמיתיים** | מבחני התכנסות, השוואה |
